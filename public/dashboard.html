Development

Aplikasi ini dikelola dan dikembangkan secara individu oleh Muhammad Ihsan, aplikasi THESIS.AI ini dirancang untuk membantu mahasiswa dalam menulis skripsi. Aplikasi ini menggunakan teknologi Node.js, Express.js, dan Socket.io untuk memberikan panduan secara real-time terkait penulisan skripsi. Aplikasi THESIS.AI ini dibuat untuk memenuhi tugas Ulangan Akhir Semester (UAS) pada mata kuliah Kecerdasan Buatan (AI).
